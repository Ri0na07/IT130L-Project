Website Name: ExploreEasy Travel and Tours & Art Collections

## Project Purpose
This website is developed for ExploreEasy Travel and Tours & Art Collections, a local business offering travel packages, tour services, and local artworks. The objective is to provide a clear, responsive, and functional platform to present the business information, services, and contact details.

## Technologies and Features Used
- HTML5 for page structure, links, lists, tables, forms, image maps, and layout
- CSS3 for styling, colors, gradients, shadows, borders, opacity, and positioning
- Bootstrap 5 for responsive design, navigation bar, dropdown menu, and image slider
- Lightbox for image preview functionality
- Responsive layout compatible with desktop, tablet, and mobile devices
- Fully tested and working on Google Chrome

---

## How to Use
1. Extract the compressed folder
2. Open the folder
3. Open index.html in any web browser
4. Navigate between pages using the main menu
